# NFT Rarity Tracker

A Solidity smart contract that stores rarity scores for NFTs.

## Features
- Rarity mapping for each token ID
- Simple getter and setter functions
- Minimal gas usage

## Technologies
Solidity  
ERC721 compatible  
Base or Ethereum
